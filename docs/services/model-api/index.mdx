# Model API

The Model API is the inference service for Bisakerja. It is responsible for generating fit scores, explainable score breakdowns, and skill gap outputs that help users decide how to approach a job opportunity.

## Source Repository And Sync Target

- Assumed source repository: `model-api`
- Source docs root: `/docs/**`
- Central sync target: `docs/services/model-api/synced/**`

## Platform Responsibilities

- Compute job fit score using job, profile, and preference context
- Generate explainable outputs for score interpretation
- Support skill gap and recommendation results returned through the backend
- Provide a stable inference boundary to the rest of the application

## Key Dependencies And Consumers

| Type                  | System                            | Why It Matters                                                               |
| --------------------- | --------------------------------- | ---------------------------------------------------------------------------- |
| Upstream consumer     | Backend API                       | The backend invokes the model layer for user-facing recommendation workflows |
| Upstream data context | User, preference, and job records | Model output depends on prepared data assembled by the backend               |
| Cross-service context | Overview docs                     | Explains why model inference is part of the product promise                  |
