## Core Flow

```text
external job platforms
  -> Scraper API
  -> raw payload storage
  -> normalization
  -> AI enrichment handoff
  -> sync preparation
  -> main Backend API database
```

Baseline daily pipeline:

```text
01:00 scrape -> 01:30 normalize -> 02:00 enrich -> 03:00 sync -> 05:00-06:00 notify
```
