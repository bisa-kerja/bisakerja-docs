## Contract Rule

Source-specific differences stop at the adapter and normalizer boundary. Backend API consumers receive normalized job records only.

Shared source HTTP behavior, including timeout, retry classification, rate limiting, backoff, and circuit breaker policy, is documented in [Job Sources](../integrations/job-sources.mdx).
