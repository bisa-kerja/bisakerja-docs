# Frontend UI

The Frontend UI is the user-facing web application for Bisakerja. It is responsible for presenting job discovery, fit analysis, preference management, and application tracking in one coherent experience.

## Source Repository And Sync Target

- Assumed source repository: `frontend-ui`
- Source docs root: `/docs/**`
- Central sync target: `docs/services/frontend-ui/synced/**`

## Platform Responsibilities

- render the job search and job detail experience
- capture user input for profile and preferences
- surface fit score, explanation, and skill gap results returned through the backend
- expose application tracking workflows to the user

## Key Dependencies And Consumers

| Type                  | System        | Why It Matters                                               |
| --------------------- | ------------- | ------------------------------------------------------------ |
| Downstream dependency | Backend API   | Provides all user-facing data and workflow actions           |
| Cross-service context | Overview docs | Explains how frontend behavior maps to platform architecture |
| Primary consumer      | End users     | The frontend is the visible product surface                  |
