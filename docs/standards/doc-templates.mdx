# Document Templates

This document standardizes the minimum template for the main document types used in the central repository.

## Common Frontmatter Template

```md
---
title: Page Title
description: One-sentence reader-facing summary.
owner: platform-docs-maintainer
reviewers:
  - engineering-lead
doc_status: active
last_reviewed: 2026-04-15
---
```

## Overview Page Template

```md
# Page Title

Short explanation of what this page covers and why it matters.

## Purpose

## Scope

## Main Content

## Related Docs
```

## Service Landing Page Template

```md
# Service Name

Short service summary.

## Source Repository And Sync Target

## Platform Responsibilities

## Key Dependencies And Consumers

## Central Docs Maintained Here

## Service-Owned Documentation Expected From Source Repo

## Pre-Sync Documentation Gap Assessment

## Related Docs
```

## Operational Page Template

```md
# Page Title

Short operational context.

## Purpose

## Workflow Or Scenario Details

## First Checks Or Action Frame

## Related Docs
```

## Standards Page Template

```md
# Standard Title

Short explanation of what the rule governs.

## Scope

## Rules

## Required Inputs Or Metadata

## Exceptions

## Related Docs
```

## Reference Page Template

```md
# Reference Title

Short explanation of the reusable concept set.

## Reference Data

## Naming Or Ownership Notes

## Related Docs
```

## Template Rule

Templates are the minimum floor, not the maximum length. Pages may add sections, but they should not remove the required structural sections without a clear reason.
