# Glossary

## Application Intelligence

The product capability that helps users track application outcomes and learn from them over time.

## Backend API

The primary application service responsible for business logic, persistence, auth, and orchestration between frontend, data, and model services.

## Explainable AI

The system behavior that explains why a score or recommendation was generated rather than returning only a raw number.

## Fit Score

A computed score indicating how well a user matches a job based on skills, experience, and preferences.

## Frontend UI

The user-facing web application through which users browse jobs, manage preferences, view scores, and track applications.

## Model API

The service responsible for returning fit scores, explanation outputs, and related AI-driven recommendation results.

## Normalized Job Record

A job listing transformed from external source format into a consistent internal data shape for the platform.

## Scraper API

The service that fetches job data from external sources and transforms it into normalized records.

## Skill Gap Analysis

The process of identifying skills that a user lacks or needs to strengthen for a target job.

## Source Of Truth

The system or repository that is considered authoritative for a particular document or dataset.
